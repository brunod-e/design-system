# Design System

This is a monorepo project created using Turbo and Changesets. This project involves the development of a Design System for the Rocketseat course.

## Instalação

Make sure to have Turbo installed globally:

```bash
npm install -g turbo
```

```bash
npm install
```

# Scripts

```bash
npm run dev
```
